0.1 RESUMEN EJECUTIVO
--------------------------------------------------------------------------------

La Asociación de Ganaderos de la Provincia de Vallegrande (AGAPROVA) agrupa a
productores dedicados a la cría y engorde de ganado bovino para su venta en pie,
valuada bajo el estándar de peso kilo gancho. La comercialización se realiza
semanalmente, constituyendo la principal fuente de ingresos de los asociados.

Actualmente, la decisión de asignación de rutas de despacho se toma de forma
empírica, basada en la experiencia individual de cada socio, lo que ha generado
pérdidas recurrentes por: elección incorrecta de ruta ante condiciones de lluvia
(especialmente en la vía Ipati-Abapó), llegada fuera del horario establecido
por los mercados (08:00 a.m.), y pérdidas por enviar ganado en condición
invernal (vaca flaca) cuando el precio de mercado no compensa el menor
rendimiento en kilo gancho.

El Sistema de Soporte a la Decisión (DSS) AGAPROVA implementa un modelo de
optimización basado en el algoritmo Simplex para maximizar la rentabilidad neta
semanal, considerando cuatro rutas de despacho, precios de mercado (Santa Cruz
y Cochabamba), eficiencia de inversión ajustada por estacionalidad y condición
del ganado, costos logísticos, bloqueos viales, y restricción climática con
umbral probabilístico en la zona de Abapó.

0.2 VARIABLES CRÍTICAS DEL SISTEMA
--------------------------------------------------------------------------------

Variables de decisión:
  x1 = Cabezas hacia Santa Cruz vía Samaipata (Ruta Principal)
  x2 = Cabezas hacia Cochabamba vía Comarapa (Ruta Principal)
  x3 = Cabezas hacia Santa Cruz vía Ipati-Abapó (Ruta Alternativa)
  x4 = Cabezas hacia Cochabamba vía Aiquile (Ruta Alternativa)

Parámetros de entrada:
  P_SC  = Precio del kilogramo gancho en Santa Cruz (Bs/kg)
  P_CB  = Precio del kilogramo gancho en Cochabamba (Bs/kg)
  C1    = Costo logístico por cabeza - Ruta Samaipata (Bs/cabeza)
  C2    = Costo logístico por cabeza - Ruta Comarapa (Bs/cabeza)
  C3    = Costo logístico por cabeza - Ruta Ipati-Abapó (Bs/cabeza)
  C4    = Costo logístico por cabeza - Ruta Aiquile (Bs/cabeza)
  G_totales = Total de cabezas disponibles
  E_inv_real = Eficiencia de inversión real (kg gancho/cabeza)
  prob_lluvia = Probabilidad de lluvia en Abapó (0.00 a 1.00)
  hora_salida = Hora de salida programada
  α, β, γ, δ = Estado de bloqueo (1=activa, 0=bloqueada)

Factores de ajuste:
  factor_estacion:  seca=1.00, lluviosa=0.70, transición=0.85
  factor_condicion: buena=1.00, regular=0.90, invernal=0.75

Variables calculadas:
  E_inv_efectiva = E_inv_real * factor_estacion * factor_condicion
  M1 = (P_SC * E_inv_efectiva) - C1
  M2 = (P_CB * E_inv_efectiva) - C2
  M3 = (P_SC * E_inv_efectiva) - C3
  M4 = (P_CB * E_inv_efectiva) - C4

Función objetivo:
  Maximizar Z = M1*x1 + M2*x2 + M3*x3 + M4*x4

Restricciones:
  (1) x1 + x2 + x3 + x4 = G_totales
  (2) x1 <= G_totales * α
  (3) x2 <= G_totales * β
  (4) x3 <= G_totales * γ
  (5) x4 <= G_totales * δ
  (6) x3 <= G_totales * Y  (Y=1 si prob_lluvia < 0.40, Y=0 si >= 0.40)
  (7) x1, x2, x3, x4 >= 0

Restricción horaria:
  Si hora_salida + tiempo_ruta(i) > 08:00, la ruta i se descarta.

Regla de asignación:
  Si tras todos los filtros existe al menos una ruta disponible,
  TODAS las cabezas se asignan a la ruta con el mayor Mi.
  Si ninguna ruta disponible, factible = false.

1.3 SCRIPT SCHEMA.SQL
--------------------------------------------------------------------------------

-- ============================================================================
-- DSS AGAPROVA - Esquema de Base de Datos
-- Version 1.0 - MySQL 8+
-- ============================================================================

CREATE DATABASE IF NOT EXISTS dss_agaprova
  CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;

USE dss_agaprova;

-- TABLAS CATALOGO

CREATE TABLE mercados (
    id INT AUTO_INCREMENT PRIMARY KEY,
    nombre VARCHAR(100) NOT NULL,
    ciudad VARCHAR(100) NOT NULL,
    activo TINYINT(1) NOT NULL DEFAULT 1,
    CONSTRAINT uq_mercados_nombre UNIQUE (nombre)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COLLATE=utf8mb4_unicode_ci;

CREATE TABLE estaciones (
    id INT AUTO_INCREMENT PRIMARY KEY,
    nombre VARCHAR(50) NOT NULL,
    factor DECIMAL(4,2) NOT NULL,
    descripcion VARCHAR(255),
    CONSTRAINT ck_estaciones_factor CHECK (factor >= 0.50 AND factor <= 1.00),
    CONSTRAINT uq_estaciones_nombre UNIQUE (nombre)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COLLATE=utf8mb4_unicode_ci;

CREATE TABLE condiciones_ganado (
    id INT AUTO_INCREMENT PRIMARY KEY,
    nombre VARCHAR(50) NOT NULL,
    factor DECIMAL(4,2) NOT NULL,
    descripcion VARCHAR(255),
    CONSTRAINT ck_condiciones_factor CHECK (factor >= 0.50 AND factor <= 1.00),
    CONSTRAINT uq_condiciones_nombre UNIQUE (nombre)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COLLATE=utf8mb4_unicode_ci;

CREATE TABLE rutas (
    id INT AUTO_INCREMENT PRIMARY KEY,
    codigo VARCHAR(10) NOT NULL,
    nombre VARCHAR(100) NOT NULL,
    origen VARCHAR(100) NOT NULL,
    destino VARCHAR(100) NOT NULL,
    mercado_id INT NOT NULL,
    tipo_via VARCHAR(50),
    tiempo_horas DECIMAL(4,1) NOT NULL,
    activo TINYINT(1) NOT NULL DEFAULT 1,
    CONSTRAINT uq_rutas_codigo UNIQUE (codigo),
    CONSTRAINT ck_rutas_tiempo CHECK (tiempo_horas > 0),
    CONSTRAINT fk_rutas_mercado FOREIGN KEY (mercado_id)
        REFERENCES mercados(id) ON DELETE RESTRICT ON UPDATE CASCADE,
    INDEX idx_rutas_mercado (mercado_id),
    INDEX idx_rutas_activo (activo)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COLLATE=utf8mb4_unicode_ci;

CREATE TABLE usuarios (
    id INT AUTO_INCREMENT PRIMARY KEY,
    username VARCHAR(50) NOT NULL,
    password_hash VARCHAR(255) NOT NULL,
    nombre VARCHAR(100) NOT NULL,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    CONSTRAINT uq_usuarios_username UNIQUE (username)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COLLATE=utf8mb4_unicode_ci;

CREATE TABLE lotes_ganado (
    id INT AUTO_INCREMENT PRIMARY KEY,
    cabezas INT NOT NULL,
    peso_promedio_kg DECIMAL(6,2) NOT NULL,
    condicion_id INT NOT NULL,
    estacion_id INT NOT NULL,
    hora_salida TIME NOT NULL DEFAULT '20:00:00',
    fecha_registro DATE NOT NULL,
    activo TINYINT(1) NOT NULL DEFAULT 1,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    CONSTRAINT ck_lotes_cabezas CHECK (cabezas > 0),
    CONSTRAINT ck_lotes_peso CHECK (peso_promedio_kg > 0),
    CONSTRAINT fk_lotes_condicion FOREIGN KEY (condicion_id)
        REFERENCES condiciones_ganado(id) ON DELETE RESTRICT ON UPDATE CASCADE,
    CONSTRAINT fk_lotes_estacion FOREIGN KEY (estacion_id)
        REFERENCES estaciones(id) ON DELETE RESTRICT ON UPDATE CASCADE,
    INDEX idx_lotes_fecha (fecha_registro),
    INDEX idx_lotes_activo (activo),
    INDEX idx_lotes_condicion (condicion_id),
    INDEX idx_lotes_estacion (estacion_id)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COLLATE=utf8mb4_unicode_ci;

CREATE TABLE precios (
    id INT AUTO_INCREMENT PRIMARY KEY,
    mercado_id INT NOT NULL,
    precio_kg DECIMAL(10,2) NOT NULL,
    fecha_registro DATE NOT NULL,
    activo TINYINT(1) NOT NULL DEFAULT 1,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    CONSTRAINT ck_precios_positivo CHECK (precio_kg > 0),
    CONSTRAINT fk_precios_mercado FOREIGN KEY (mercado_id)
        REFERENCES mercados(id) ON DELETE RESTRICT ON UPDATE CASCADE,
    INDEX idx_precios_fecha (fecha_registro),
    INDEX idx_precios_activo (activo),
    INDEX idx_precios_mercado_fecha (mercado_id, fecha_registro)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COLLATE=utf8mb4_unicode_ci;

CREATE TABLE clima (
    id INT AUTO_INCREMENT PRIMARY KEY,
    probabilidad_lluvia DECIMAL(4,2) NOT NULL,
    ubicacion VARCHAR(100) NOT NULL DEFAULT 'Abapo',
    fecha_registro DATE NOT NULL,
    activo TINYINT(1) NOT NULL DEFAULT 1,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    CONSTRAINT ck_clima_probabilidad CHECK (
        probabilidad_lluvia >= 0 AND probabilidad_lluvia <= 1
    ),
    INDEX idx_clima_fecha (fecha_registro),
    INDEX idx_clima_activo (activo)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COLLATE=utf8mb4_unicode_ci;

CREATE TABLE bloqueos (
    id INT AUTO_INCREMENT PRIMARY KEY,
    ruta_id INT NOT NULL,
    activo TINYINT(1) NOT NULL DEFAULT 0,
    fecha_inicio DATE,
    fecha_fin DATE,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    CONSTRAINT fk_bloqueos_ruta FOREIGN KEY (ruta_id)
        REFERENCES rutas(id) ON DELETE RESTRICT ON UPDATE CASCADE,
    INDEX idx_bloqueos_ruta (ruta_id),
    INDEX idx_bloqueos_activo (activo)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COLLATE=utf8mb4_unicode_ci;

CREATE TABLE costos_flete (
    id INT AUTO_INCREMENT PRIMARY KEY,
    ruta_id INT NOT NULL,
    costo_cabeza DECIMAL(10,2) NOT NULL,
    semana_inicio DATE NOT NULL,
    activo TINYINT(1) NOT NULL DEFAULT 1,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    CONSTRAINT ck_costos_positivo CHECK (costo_cabeza > 0),
    CONSTRAINT fk_costos_ruta FOREIGN KEY (ruta_id)
        REFERENCES rutas(id) ON DELETE RESTRICT ON UPDATE CASCADE,
    INDEX idx_costos_ruta (ruta_id),
    INDEX idx_costos_activo (activo),
    INDEX idx_costos_semana (semana_inicio)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COLLATE=utf8mb4_unicode_ci;

CREATE TABLE escenarios_simulacion (
    id INT AUTO_INCREMENT PRIMARY KEY,
    codigo VARCHAR(10) NOT NULL,
    nombre VARCHAR(100) NOT NULL,
    descripcion TEXT,
    datos_json JSON NOT NULL,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    CONSTRAINT uq_escenarios_codigo UNIQUE (codigo)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COLLATE=utf8mb4_unicode_ci;

1.4 VISTA V_SEMANA_OPERATIVA
--------------------------------------------------------------------------------

CREATE OR REPLACE VIEW v_semana_operativa AS
SELECT
    l.id AS lote_id,
    l.cabezas,
    l.peso_promedio_kg,
    l.hora_salida,
    l.fecha_registro,
    cg.nombre AS condicion,
    cg.factor AS factor_condicion,
    e.nombre AS estacion,
    e.factor AS factor_estacion,
    r.id AS ruta_id,
    r.codigo AS ruta_codigo,
    r.nombre AS ruta_nombre,
    r.tiempo_horas,
    r.tipo_via,
    m.id AS mercado_id,
    m.nombre AS mercado_nombre,
    m.ciudad AS mercado_ciudad,
    p.precio_kg,
    cf.costo_cabeza,
    COALESCE(
        (SELECT b.activo FROM bloqueos b
         WHERE b.ruta_id = r.id AND b.activo = 1
         LIMIT 1),
        0
    ) AS bloqueado,
    cl.probabilidad_lluvia,
    l.peso_promedio_kg * cg.factor * e.factor AS eficiencia_inv_efectiva,
    CASE
        WHEN cl.probabilidad_lluvia >= 0.40 THEN 0
        ELSE 1
    END AS clima_permite_ruta3
FROM lotes_ganado l
JOIN condiciones_ganado cg ON l.condicion_id = cg.id
JOIN estaciones e ON l.estacion_id = e.id
CROSS JOIN rutas r
JOIN mercados m ON r.mercado_id = m.id
LEFT JOIN precios p ON p.mercado_id = m.id AND p.activo = 1
LEFT JOIN costos_flete cf ON cf.ruta_id = r.id AND cf.activo = 1
LEFT JOIN clima cl ON cl.activo = 1
WHERE l.activo = 1
  AND r.activo = 1;

Nota: La vista usa una subconsulta correlacionada para obtener el estado de
bloqueo activo (COALESCE + SELECT ... LIMIT 1), lo que evita la duplicacion
de filas cuando hay multiples registros de bloqueo para una misma ruta.
La probabilidad de lluvia se obtiene del unico registro climatico activo
(solo debe haber uno activo a la vez).

2.4 ROUTER (app/Core/Router.php)
--------------------------------------------------------------------------------

<?php
namespace App\Core;

class Router
{
    private array $routes = [];

    public function get(string $path, string $handler, array $middleware = []): void
    {
        $this->addRoute('GET', $path, $handler, $middleware);
    }

    public function post(string $path, string $handler, array $middleware = []): void
    {
        $this->addRoute('POST', $path, $handler, $middleware);
    }

    private function addRoute(string $method, string $path, string $handler, array $middleware): void
    {
        $pattern = preg_replace('/\{([a-zA-Z_]+)\}/', '(?P<$1>[^/]+)', $path);
        $pattern = '#^' . $pattern . '$#';
        $this->routes[] = [
            'method' => $method,
            'pattern' => $pattern,
            'handler' => $handler,
            'middleware' => $middleware
        ];
    }

    public function resolve(string $method, string $uri): mixed
    {
        $uri = parse_url($uri, PHP_URL_PATH);
        $uri = rtrim($uri, '/') ?: '/';
        $uri = urldecode($uri);

        foreach ($this->routes as $route) {
            if ($route['method'] !== $method) continue;
            if (preg_match($route['pattern'], $uri, $matches)) {
                $params = array_filter($matches, 'is_string', ARRAY_FILTER_USE_KEY);

                foreach ($route['middleware'] as $mw) {
                    $mwClass = 'App\Core\Middleware\' . $mw;
                    if (class_exists($mwClass)) (new $mwClass())->handle();
                }

                [$controllerClass, $methodName] = explode('@', $route['handler']);
                $controllerClass = 'App\Controllers\' . $controllerClass;

                if (!class_exists($controllerClass)) {
                    throw new \RuntimeException("Controller {$controllerClass} not found");
                }

                $controller = new $controllerClass();
                return $controller->$methodName(...$params);
            }
        }

        http_response_code(404);
        require __DIR__ . '/../Views/errors/404.php';
        return null;
    }
}

<?php
namespace App\Core;

class Request
{
    private array $get;
    private array $post;
    private array $server;
    private array $files;
    private array $cookies;

    public function __construct()
    {
        $this->get = $_GET;
        $this->post = $_POST;
        $this->server = $_SERVER;
        $this->files = $_FILES;
        $this->cookies = $_COOKIE;
    }

    public function get(string $key, mixed $default = null): mixed
    {
        return $this->get[$key] ?? $default;
    }

    public function post(string $key, mixed $default = null): mixed
    {
        return $this->post[$key] ?? $default;
    }

    public function all(): array
    {
        return $this->post;
    }

    public function server(string $key, mixed $default = null): mixed
    {
        return $this->server[$key] ?? $default;
    }

    public function method(): string
    {
        return strtoupper($this->server('REQUEST_METHOD', 'GET'));
    }

    public function uri(): string
    {
        $uri = $this->server('REQUEST_URI', '/');
        $basePath = dirname($this->server('SCRIPT_NAME'));
        if ($basePath !== '/' && strpos($uri, $basePath) === 0) {
            $uri = substr($uri, strlen($basePath));
        }
        return $uri ?: '/';
    }

    public function baseUrl(): string
    {
        $scheme = (!empty($this->server('HTTPS')) && $this->server('HTTPS') !== 'off') ? 'https' : 'http';
        $host = $this->server('HTTP_HOST');
        return $scheme . '://' . $host;
    }

    public function isPost(): bool
    {
        return $this->method() === 'POST';
    }

    public function file(string $key): ?array
    {
        return $this->files[$key] ?? null;
    }

    public function csrfToken(): string
    {
        return Session::get('csrf_token');
    }

    public function verifyCsrf(): bool
    {
        $token = $this->post('_csrf_token');
        if (!$token) return false;
        return hash_equals(Session::get('csrf_token'), $token);
    }
}

<?php
namespace App\Core;

class Session
{
    private static bool $started = false;

    public static function start(): void
    {
        if (!self::$started && session_status() === PHP_SESSION_NONE) {
            session_start();
            self::$started = true;
            if (!isset($_SESSION['_last_regenerate'])) {
                $_SESSION['_last_regenerate'] = time();
            } elseif (time() - $_SESSION['_last_regenerate'] > SESSION_LIFETIME) {
                session_regenerate_id(true);
                $_SESSION['_last_regenerate'] = time();
            }
            if (!isset($_SESSION['csrf_token'])) {
                $_SESSION['csrf_token'] = bin2hex(random_bytes(32));
            }
        }
    }

    public static function get(string $key, mixed $default = null): mixed
    {
        self::start();
        return $_SESSION[$key] ?? $default;
    }

    public static function set(string $key, mixed $value): void
    {
        self::start();
        $_SESSION[$key] = $value;
    }

    public static function remove(string $key): void
    {
        self::start();
        unset($_SESSION[$key]);
    }

    public static function destroy(): void
    {
        self::start();
        $_SESSION = [];
        if (ini_get('session.use_cookies')) {
            $params = session_get_cookie_params();
            setcookie(session_name(), '', time() - 42000,
                $params['path'], $params['domain'],
                $params['secure'], $params['httponly']);
        }
        session_destroy();
        self::$started = false;
    }

    public static function flash(string $key, string $message): void
    {
        self::set('_flash_' . $key, $message);
    }

    public static function getFlash(string $key, mixed $default = null): mixed
    {
        $value = self::get('_flash_' . $key, $default);
        self::remove('_flash_' . $key);
        return $value;
    }

    public static function hasFlash(string $key): bool
    {
        return isset($_SESSION['_flash_' . $key]);
    }
}

2.9 MODEL (app/Core/Model.php) - Active Record base
--------------------------------------------------------------------------------

<?php
namespace App\Core;

abstract class Model
{
    protected static string $table = '';
    protected static string $primaryKey = 'id';
    protected array $attributes = [];
    protected array $original = [];
    protected bool $exists = false;

    public function __construct(array $attributes = [])
    {
        $this->fill($attributes);
    }

    public function fill(array $attributes): void
    {
        foreach ($attributes as $key => $value) {
            $this->attributes[$key] = $value;
        }
    }

    public static function all(): array
    {
        $db = Database::getInstance();
        $stmt = $db->query("SELECT * FROM " . static::$table . " ORDER BY id DESC");
        return static::hydrate($stmt->fetchAll());
    }

    public static function find(int|string $id): ?static
    {
        $db = Database::getInstance();
        $stmt = $db->query(
            "SELECT * FROM " . static::$table . " WHERE " . static::$primaryKey . " = ?", [$id]);
        $data = $stmt->fetch();
        if (!$data) return null;
        $model = new static();
        $model->attributes = $data;
        $model->original = $data;
        $model->exists = true;
        return $model;
    }

    public static function where(string $column, mixed $value, string $operator = '='): array
    {
        $db = Database::getInstance();
        $stmt = $db->query(
            "SELECT * FROM " . static::$table . " WHERE {$column} {$operator} ? ORDER BY id DESC", [$value]);
        return static::hydrate($stmt->fetchAll());
    }

    public static function firstWhere(string $column, mixed $value, string $operator = '='): ?static
    {
        $results = static::where($column, $value, $operator);
        return $results[0] ?? null;
    }

    public function save(): bool
    {
        $db = Database::getInstance();
        if ($this->exists) {
            $sets = []; $params = [];
            foreach ($this->attributes as $key => $value) {
                if ($key === static::$primaryKey) continue;
                if ($value !== ($this->original[$key] ?? null)) {
                    $sets[] = "{$key} = ?"; $params[] = $value;
                }
            }
            if (empty($sets)) return true;
            $params[] = $this->attributes[static::$primaryKey];
            $sql = "UPDATE " . static::$table . " SET " . implode(', ', $sets)
                 . " WHERE " . static::$primaryKey . " = ?";
            $db->execute($sql, $params);
            $this->original = $this->attributes;
            return true;
        } else {
            $columns = array_keys($this->attributes);
            $placeholders = array_fill(0, count($columns), '?');
            $sql = "INSERT INTO " . static::$table . " (" . implode(', ', $columns) . ")"
                 . " VALUES (" . implode(', ', $placeholders) . ")";
            $db->execute($sql, array_values($this->attributes));
            $this->attributes[static::$primaryKey] = $db->lastInsertId();
            $this->original = $this->attributes;
            $this->exists = true;
            return true;
        }
    }

    public function delete(): bool
    {
        if (!$this->exists) return false;
        $db = Database::getInstance();
        $db->execute("DELETE FROM " . static::$table . " WHERE " . static::$primaryKey . " = ?",
            [$this->attributes[static::$primaryKey]]);
        $this->exists = false;
        return true;
    }

    public function __get(string $name): mixed { return $this->attributes[$name] ?? null; }
    public function __set(string $name, mixed $value): void { $this->attributes[$name] = $value; }
    public function __isset(string $name): bool { return isset($this->attributes[$name]); }
    public function toArray(): array { return $this->attributes; }

    protected static function hydrate(array $records): array
    {
        $models = [];
        foreach ($records as $data) {
            $model = new static();
            $model->attributes = $data;
            $model->original = $data;
            $model->exists = true;
            $models[] = $model;
        }
        return $models;
    }

    public static function count(): int
    {
        $db = Database::getInstance();
        $stmt = $db->query("SELECT COUNT(*) as total FROM " . static::$table);
        return (int)$stmt->fetch()['total'];
    }
}

2.10 CONTROLLER (app/Core/Controller.php)

<?php
namespace App\Core;

abstract class Controller
{
    protected Request $request;
    protected Response $response;
    protected Validator $validator;

    public function __construct()
    {
        $this->request = new Request();
        $this->response = new Response();
        $this->validator = new Validator();
    }

    protected function view(string $view, array $data = []): void { Response::view($view, $data); }
    protected function redirect(string $url): void { Response::redirect($url); }
    protected function json(mixed $data, int $statusCode = 200): void { Response::json($data, $statusCode); }

    protected function validated(array $rules): array
    {
        $data = $this->request->all();
        $this->validator->validate($data, $rules);
        if ($this->validator->hasErrors()) {
            Session::flash('errors', $this->validator->getErrors());
            Session::flash('old', $data);
            Response::redirect($this->request->server('HTTP_REFERER', '/'));
        }
        return $data;
    }

    protected function verifyCsrfOrFail(): void
    {
        if (!$this->request->verifyCsrf()) {
            Session::flash('error', 'Token CSRF invalido. Intente nuevamente.');
            Response::redirect($this->request->server('HTTP_REFERER', '/'));
        }
    }
}

<?php
namespace App\Core;

class Validator
{
    private array $errors = [];

    public function validate(array $data, array $rules): bool
    {
        $this->errors = [];
        foreach ($rules as $field => $ruleSet) {
            $ruleList = is_array($ruleSet) ? $ruleSet : explode('|', $ruleSet);
            $value = $data[$field] ?? null;
            foreach ($ruleList as $rule) {
                $params = [];
                if (str_contains($rule, ':')) {
                    [$rule, $paramStr] = explode(':', $rule, 2);
                    $params = explode(',', $paramStr);
                }
                $methodName = 'rule' . ucfirst($rule);
                if (method_exists($this, $methodName)) {
                    $this->$methodName($field, $value, $params);
                }
            }
        }
        return empty($this->errors);
    }

    public function getErrors(): array { return $this->errors; }
    public function hasErrors(): bool { return !empty($this->errors); }

    private function ruleRequired(string $field, mixed $value, array $params): void
    {
        if ($value === null || $value === '' || $value === [])
            $this->errors[$field][] = 'El campo es obligatorio.';
    }

    private function ruleNumeric(string $field, mixed $value, array $params): void
    {
        if ($value !== null && $value !== '' && !is_numeric($value))
            $this->errors[$field][] = 'Debe ser un valor numerico.';
    }

    private function ruleInteger(string $field, mixed $value, array $params): void
    {
        if ($value !== null && $value !== '' && filter_var($value, FILTER_VALIDATE_INT) === false)
            $this->errors[$field][] = 'Debe ser un numero entero.';
    }

    private function ruleMin(string $field, mixed $value, array $params): void
    {
        $min = (float)$params[0];
        if (is_numeric($value) && (float)$value < $min)
            $this->errors[$field][] = "El valor minimo es {$min}.";
    }

    private function ruleMax(string $field, mixed $value, array $params): void
    {
        $max = (float)$params[0];
        if (is_numeric($value) && (float)$value > $max)
            $this->errors[$field][] = "El valor maximo es {$max}.";
    }

    private function ruleIn(string $field, mixed $value, array $params): void
    {
        if ($value !== null && $value !== '' && !in_array((string)$value, $params))
            $this->errors[$field][] = 'El valor seleccionado no es valido.';
    }
}

<?php
namespace App\Controllers;

use App\Core\Controller;
use App\Core\Database;
use App\Core\Response;
use App\Core\Session;
use App\Services\SimplexSolver;

class SimplexController extends Controller

    private SimplexSolver $simplexSolver;

    public function __construct()
    {
        parent::__construct();
        $this->simplexSolver = new SimplexSolver();
    }

    public function index(): void
    {
        $db = Database::getInstance();
        $lote = $db->query(
            "SELECT l.*, c.nombre AS condicion, c.factor AS factor_condicion,
                    e.nombre AS estacion, e.factor AS factor_estacion
             FROM lotes_ganado l
             JOIN condiciones_ganado c ON l.condicion_id = c.id
             JOIN estaciones e ON l.estacion_id = e.id
             WHERE l.activo = 1 ORDER BY l.id DESC LIMIT 1"
        )->fetch();
        $precios = $db->query("SELECT p.*, m.id AS mercado_id_ref FROM precios p JOIN mercados m ON p.mercado_id = m.id WHERE p.activo = 1")->fetchAll();
        $costos = $db->query("SELECT cf.*, r.codigo FROM costos_flete cf JOIN rutas r ON cf.ruta_id = r.id WHERE cf.activo = 1")->fetchAll();
        $clima = $db->query("SELECT * FROM clima WHERE activo = 1 ORDER BY id DESC LIMIT 1")->fetch();
        $bloqueos = $db->query("SELECT b.*, r.codigo FROM bloqueos b JOIN rutas r ON b.ruta_id = r.id WHERE b.activo = 1")->fetchAll();

        $this->view('simplex/resultado', [
            'lote' => $lote, 'precios' => $precios, 'costos' => $costos,
            'clima' => $clima, 'bloqueos' => $bloqueos, 'resultado' => null
        ]);
    }

    public function calcular(): void
    {
        $this->verifyCsrfOrFail();
        $db = Database::getInstance();

        $lote = $db->query(
            "SELECT l.*, c.factor AS factor_condicion, e.factor AS factor_estacion
             FROM lotes_ganado l
             JOIN condiciones_ganado c ON l.condicion_id = c.id
             JOIN estaciones e ON l.estacion_id = e.id
             WHERE l.activo = 1 ORDER BY l.id DESC LIMIT 1"
        )->fetch();
        if (!$lote) { Session::flash('error', 'No hay lotes registrados.'); Response::redirect('/lotes/crear'); }

        $precios = $db->query("SELECT p.*, m.id AS mercado_id_ref FROM precios p JOIN mercados m ON p.mercado_id = m.id WHERE p.activo = 1")->fetchAll();
        if (count($precios) < 2) { Session::flash('error', 'Registre precios para ambos mercados.'); Response::redirect('/precios/crear'); }

        $costos = $db->query("SELECT cf.*, r.codigo FROM costos_flete cf JOIN rutas r ON cf.ruta_id = r.id WHERE cf.activo = 1")->fetchAll();
        if (count($costos) < 4) { Session::flash('error', 'Registre costos para las 4 rutas.'); Response::redirect('/costos/crear'); }

        $clima = $db->query("SELECT * FROM clima WHERE activo = 1 ORDER BY id DESC LIMIT 1")->fetch();
        $bloqueos = $db->query("SELECT b.*, r.codigo FROM bloqueos b JOIN rutas r ON b.ruta_id = r.id WHERE b.activo = 1")->fetchAll();

        // Precios por mercado
        $precioSC = $precioCB = 0;
        foreach ($precios as $p) {
            if ($p['mercado_id_ref'] == 1) $precioSC = (float)$p['precio_kg'];
            if ($p['mercado_id_ref'] == 2) $precioCB = (float)$p['precio_kg'];
        }

        // Costos por ruta
        $costosPorRuta = [];
        foreach ($costos as $c) $costosPorRuta[$c['codigo']] = (float)$c['costo_cabeza'];

        // Bloqueos activos
        $bloqueosActivos = [];
        foreach ($bloqueos as $b) $bloqueosActivos[$b['codigo']] = true;

        // E_inv_efectiva
        $eInvEfectiva = (float)$lote['peso_promedio_kg'] * (float)$lote['factor_estacion'] * (float)$lote['factor_condicion'];

        // Calcular Mi
        $Mi = [
            'R01' => ($precioSC * $eInvEfectiva) - ($costosPorRuta['R01'] ?? 0),
            'R02' => ($precioCB * $eInvEfectiva) - ($costosPorRuta['R02'] ?? 0),
            'R03' => ($precioSC * $eInvEfectiva) - ($costosPorRuta['R03'] ?? 0),
            'R04' => ($precioCB * $eInvEfectiva) - ($costosPorRuta['R04'] ?? 0),
        ];
        $tiempos = ['R01' => 6.5, 'R02' => 9.0, 'R03' => 11.0, 'R04' => 10.0];

        $resultado = $this->simplexSolver->resolver([
            'total_cabezas' => (int)$lote['cabezas'],
            'Mi' => $Mi,
            'bloqueos' => $bloqueosActivos,
            'tiempos' => $tiempos,
            'hora_salida' => $lote['hora_salida'],
            'prob_lluvia' => $clima ? (float)$clima['probabilidad_lluvia'] : 0
        ]);

        Session::set('resultado_simplex', $resultado);

        $this->view('simplex/resultado', [
            'lote' => $lote, 'precios' => $precios, 'costos' => $costos,
            'clima' => $clima, 'bloqueos' => $bloqueos,
            'resultado' => $resultado, 'Mi' => $Mi, 'eInvEfectiva' => $eInvEfectiva
        ]);
    }
}

        if ($bloqueoR2) $bloqueos['R02'] = true;
        if ($bloqueoR3) $bloqueos['R03'] = true;
        if ($bloqueoR4) $bloqueos['R04'] = true;

        ]);

5.1 SIMPLEXSOLVER (app/Services/SimplexSolver.php)
--------------------------------------------------------------------------------

<?php
namespace App\Services;

class SimplexSolver
{
    private array $rutas = ['R01', 'R02', 'R03', 'R04'];
    private array $tiemposBase = ['R01' => 6.5, 'R02' => 9.0, 'R03' => 11.0, 'R04' => 10.0];
    private const UMBRAL_LLUVIA = 0.40;
    private const HORA_LIMITE = '08:00';

    /**
     * Metodo principal de resolucion del modelo Simplex.
     * Aplica todos los filtros (bloqueos, clima, horario) y asigna
     * todas las cabezas a la ruta con mayor margen neto (Mi) disponible.
     *
     * @param array $params {
     *   total_cabezas: int,
     *   Mi: array<string, float>  // M1..M4 por codigo de ruta
     *   bloqueos: array<string, bool>,
     *   tiempos: array<string, float>,
     *   hora_salida: string (H:i:s o H:i),
     *   prob_lluvia: float (0..1)
     * }
     * @return array Resultado con asignacion, Z, factibilidad, detalle
     */
    public function resolver(array $params): array
    {
        $totalCabezas = (int)$params['total_cabezas'];
        $Mi = $params['Mi'];
        $bloqueos = $params['bloqueos'] ?? [];
        $tiempos = $params['tiempos'] ?? $this->tiemposBase;
        $horaSalida = $params['hora_salida'] ?? '20:00';
        $probLluvia = (float)($params['prob_lluvia'] ?? 0);

        $resultado = [
            'factible' => true,
            'Z' => 0,
            'asignacion' => [],
            'ruta_elegida' => null,
            'detalle_rutas' => [],
            'mensaje' => '',
            'filtros_aplicados' => []
        ];

        // PASO 1: Evaluar restricciones para cada ruta
        $rutasDisponibles = [];

        foreach ($this->rutas as $codigo) {
            $filtros = [
                'bloqueo' => false,
                'clima' => false,
                'horario' => false
            ];
            $descartada = false;
            $motivo = '';

            // Filtro de bloqueo vial: alpha, beta, gamma, delta
            if (isset($bloqueos[$codigo]) && $bloqueos[$codigo]) {
                $filtros['bloqueo'] = true;
                $descartada = true;
                $motivo .= 'Ruta bloqueada. ';
            }

            // Filtro climatico: solo para R03 (Ipati-Abapo)
            if ($codigo === 'R03' && $probLluvia >= self::UMBRAL_LLUVIA) {
                $filtros['clima'] = true;
                $descartada = true;
                $motivo .= 'Clima adverso (prob_lluvia=' . $probLluvia . '). ';
            }

            // Filtro horario: hora_salida + tiempo_viaje <= 08:00
            $tiempoRuta = $tiempos[$codigo] ?? $this->tiemposBase[$codigo];
            $horaLlegada = $this->calcularHoraLlegada($horaSalida, $tiempoRuta);
            if ($horaLlegada > self::HORA_LIMITE) {
                $filtros['horario'] = true;
                $descartada = true;
                $motivo .= "Llegada {$horaLlegada} excede limite 08:00. ";
            }

            $resultado['detalle_rutas'][$codigo] = [
                'Mi' => $Mi[$codigo] ?? 0,
                'disponible' => !$descartada,
                'motivo_descarte' => $descartada ? trim($motivo) : '',
                'filtros' => $filtros,
                'hora_llegada' => $horaLlegada,
                'tiempo_viaje' => $tiempoRuta,
                'cabezas_asignadas' => 0
            ];

            if (!$descartada) {
                $rutasDisponibles[$codigo] = $Mi[$codigo] ?? PHP_FLOAT_MIN;
            }
        }

        $resultado['filtros_aplicados'] = [
            'bloqueos_activos' => $bloqueos,
            'prob_lluvia' => $probLluvia,
            'umbral_clima' => self::UMBRAL_LLUVIA,
            'hora_salida' => $horaSalida,
            'hora_limite' => self::HORA_LIMITE
        ];

        // PASO 2: Verificar factibilidad
        if (empty($rutasDisponibles)) {
            $resultado['factible'] = false;
            $resultado['mensaje'] = 'NO FACTIBLE: Ninguna ruta disponible. Revise bloqueos, clima y horario.';
            return $resultado;
        }

        // PASO 3: Asignar TODAS las cabezas a la ruta con mayor Mi
        arsort($rutasDisponibles);
        $rutaElegida = array_key_first($rutasDisponibles);
        $mejorMi = $rutasDisponibles[$rutaElegida];

        $resultado['ruta_elegida'] = $rutaElegida;
        $resultado['asignacion'][$rutaElegida] = $totalCabezas;
        $resultado['detalle_rutas'][$rutaElegida]['cabezas_asignadas'] = $totalCabezas;
        $resultado['Z'] = $mejorMi * $totalCabezas;

        $nombresRutas = [
            'R01' => 'Santa Cruz via Samaipata',
            'R02' => 'Cochabamba via Comarapa',
            'R03' => 'Santa Cruz via Ipati-Abapo',
            'R04' => 'Cochabamba via Aiquile'
        ];

        $resultado['mensaje'] = sprintf(
            'ASIGNACION OPTIMA: Enviar %d cabezas a %s (Margen neto: Bs %s por cabeza, Z total: Bs %s).',
            $totalCabezas,
            $nombresRutas[$rutaElegida] ?? $rutaElegida,
            number_format($mejorMi, 2, ',', '.'),
            number_format($resultado['Z'], 2, ',', '.')
        );

        $resultado['detalle'] = [
            'total_cabezas' => $totalCabezas,
            'ruta_elegida_nombre' => $nombresRutas[$rutaElegida] ?? $rutaElegida,
            'margen_unitario' => $mejorMi,
            'z_total' => $resultado['Z'],
            'metodo' => 'Asignacion total a la ruta con mayor Mi entre las disponibles'
        ];

        return $resultado;
    }

    /**
     * Calcula la hora de llegada sumando horas a la hora de salida
     */
    private function calcularHoraLlegada(string $horaSalida, float $horasViaje): string
    {
        $parts = explode(':', $horaSalida);
        $h = (int)($parts[0] ?? 20);
        $m = (int)($parts[1] ?? 0);
        $totalMinutos = $h * 60 + $m + (int)($horasViaje * 60);
        $totalMinutos = $totalMinutos % (24 * 60); // Ciclo de 24h
        return sprintf('%02d:%02d', intdiv($totalMinutos, 60), $totalMinutos % 60);
    }

    /**
     * Comparar dos horas en formato HH:MM
     */
    private function compararHoras(string $h1, string $h2): int
    {
        $t1 = strtotime($h1);
        $t2 = strtotime($h2);
        if ($t1 === false || $t2 === false) return 0;
        // Si la llegada es al dia siguiente (h1 < h2 cuando h1 deberia ser mayor)
        // Ajustar: si la diferencia es negativa, sumar 24h
        $diff = $t1 - $t2;
        if ($diff < 0 && $h1 < $h2) $diff += 86400; // 24h en segundos
        return $diff <=> 0;
    }
}

<?php
namespace App\Services;

class ClimaService
{
    private const UMBRAL_LLUVIA_ABAPO = 0.40;

    /**
     * Obtener el registro climatico activo
     */
    public function getClimaActivo(): ?array
    {
        return Database::getInstance()->query(
            "SELECT * FROM clima WHERE activo = 1 ORDER BY id DESC LIMIT 1"
        )->fetch() ?: null;
    }

    /**
     * Evaluar si la ruta Ipati-Abapo (R03) esta disponible segun el clima
     */
    public function rutaIpatiDisponible(): bool
    {
        $clima = $this->getClimaActivo();
        if (!$clima) return true; // Sin datos climaticos, se asume disponible
        return (float)$clima['probabilidad_lluvia'] < self::UMBRAL_LLUVIA_ABAPO;
    }

    /**
     * Obtener probabilidad de lluvia actual
     */
    public function getProbabilidadLluvia(): float
    {
        $clima = $this->getClimaActivo();
        return $clima ? (float)$clima['probabilidad_lluvia'] : 0.0;
    }

    /**
     * Obtener alerta si corresponde
     */
    public function getAlerta(): ?string
    {
        $clima = $this->getClimaActivo();
        if (!$clima) return null;
        $prob = (float)$clima['probabilidad_lluvia'];
        if ($prob >= self::UMBRAL_LLUVIA_ABAPO) {
            return 'ALERTA CLIMATICA: Probabilidad de lluvia en Abapo del '
                 . ($prob * 100) . '%. La ruta Ipati-Abapo (R03) esta bloqueada.';
        }
        if ($prob >= 0.25) {
            return 'PRECAUCION: Probabilidad de lluvia en Abapo del '
                 . ($prob * 100) . '%. Monitoree condiciones.';
        }
        return null;
    }
}

5.3 ESTACIONALIDADSERVICE (app/Services/EstacionalidadService.php)
--------------------------------------------------------------------------------

<?php
namespace App\Services;

use App\Core\Database;

class EstacionalidadService
{
    private array $factoresEstacion = [];
    private array $factoresCondicion = [];

    /**
     * Obtener factor de ajuste por estacion
     */
    public function getFactorEstacion(int $estacionId): float
    {
        if (empty($this->factoresEstacion)) {
            $this->cargarFactores();
        }
        return $this->factoresEstacion[$estacionId] ?? 1.00;
    }

    /**
     * Obtener factor de ajuste por condicion del ganado
     */
    public function getFactorCondicion(int $condicionId): float
    {
        if (empty($this->factoresCondicion)) {
            $this->cargarFactores();
        }
        return $this->factoresCondicion[$condicionId] ?? 1.00;
    }

    /**
     * Calcular eficiencia de inversion efectiva
     */
    public function calcularEInvEfectiva(
        float $eInvReal,
        int $estacionId,
        int $condicionId
    ): float {
        return $eInvReal
            * $this->getFactorEstacion($estacionId)
            * $this->getFactorCondicion($condicionId);
    }

    /**
     * Cargar factores desde la base de datos
     */
    private function cargarFactores(): void
    {
        $db = Database::getInstance();

        $estaciones = $db->query("SELECT id, factor FROM estaciones")->fetchAll();
        foreach ($estaciones as $e) {
            $this->factoresEstacion[(int)$e['id']] = (float)$e['factor'];
        }

        $condiciones = $db->query("SELECT id, factor FROM condiciones_ganado")->fetchAll();
        foreach ($condiciones as $c) {
            $this->factoresCondicion[(int)$c['id']] = (float)$c['factor'];
        }
    }

    /**
     * Obtener el nombre de la estacion actual basada en el mes
     */
    public function getEstacionActual(): string
    {
        $mes = (int)date('n');
        // Seca: mayo-agosto (5-8)
        // Lluviosa: diciembre-marzo (12, 1-3)
        // Transicion: abril, septiembre-noviembre (4, 9-11)
        if ($mes >= 5 && $mes <= 8) return 'Seca';
        if ($mes >= 12 || $mes <= 3) return 'Lluviosa';
        return 'Transicion';
    }

    /**
     * Obtener descripcion de la temporada
     */
    public function getDescripcionTemporada(): string
    {

            'Transicion' => 'Temporada de transicion. Condiciones variables.'
        ];
        return $descripciones[$estacion] ?? '';
    }

use Dompdf\Options;

class ReporteService

    private string $storagePath;

    public function __construct()
    {
        $this->storagePath = __DIR__ . '/../../storage/reportes/';
        if (!is_dir($this->storagePath)) {
            mkdir($this->storagePath, 0755, true);
        }
    }

    /**
     * Generar PDF del reporte semanal de decision
     *
     * @param array $data Datos para el reporte (lote, precios, clima, bloqueos, resultado)
     * @return string Ruta del archivo PDF generado
     */
    public function generarPDF(array $data): string
    {
        $options = new Options();
        $options->set('defaultFont', 'DejaVu Sans');
        $options->set('isRemoteEnabled', true);
        $options->set('isHtml5ParserEnabled', true);

        $dompdf = new Dompdf($options);

        $html = $this->generarHTMLReporte($data);

        $dompdf->loadHtml($html, 'UTF-8');
        $dompdf->setPaper('A4', 'portrait');
        $dompdf->render();

        $filename = 'reporte_semanal_' . date('Ymd_His') . '.pdf';
        $filepath = $this->storagePath . $filename;

        file_put_contents($filepath, $dompdf->output());

        return $filepath;
    }

7.1 PRUEBA DEL CASO BASE (60 cabezas, bloqueo x2)
--------------------------------------------------------------------------------

Escenario simulado:
  Stock: G_totales = 60 cabezas
  Precio Santa Cruz (P_SC): 32 Bs/kg
  Precio Cochabamba (P_CB): 34 Bs/kg
  E_inv_real: 0.58 kg gancho/cabeza
  Estacion: Seca (factor = 1.00)
  Condicion: Buena (factor = 1.00)
  E_inv_efectiva = 0.58 * 1.00 * 1.00 = 0.58

  Costos:
    C1 (Samaipata) = 420 Bs/cabeza
    C2 (Comarapa) = 510 Bs/cabeza
    C3 (Ipati-Abapo) = 390 Bs/cabeza
    C4 (Aiquile) = 480 Bs/cabeza

  Bloqueos: x2 (Comarapa, R02) BLOQUEADA (beta = 0)
  Clima: Sin alerta (prob_lluvia < 0.40)

Calculo de margenes:
  M1 = (32 * 0.58) - 420 = 18.56 - 420 = -401.44
  M2 = (34 * 0.58) - 510 = 19.72 - 510 = -490.28 (bloqueada)
  M3 = (32 * 0.58) - 390 = 18.56 - 390 = -371.44
  M4 = (34 * 0.58) - 480 = 19.72 - 480 = -460.28

Filtros:
  R01 (Samaipata): Disponible (alpha=1, horario ok: 20:00+6.5h=02:30 <= 08:00)
  R02 (Comarapa): BLOQUEADA (beta=0)
  R03 (Ipati-Abapo): Disponible (gamma=1, clima ok, horario ok: 20:00+11h=07:00 <= 08:00)
  R04 (Aiquile): Disponible (delta=1, horario ok: 20:00+10h=06:00 <= 08:00)

Rutas disponibles: R01 (M1=-401.44), R03 (M3=-371.44), R04 (M4=-460.28)
Mejor Mi: M3 = -371.44 (Ruta Ipati-Abapo, menos negativa)

Asignacion: x3 = 60 cabezas
Z = 60 * (-371.44) = -22,286.40 Bs

Resultado esperado del algoritmo SimplexSolver:
  factible: true
  ruta_elegida: 'R03'
  Z: -22286.40
  asignacion: { 'R03': 60 }
  mensaje debe indicar que R03 fue seleccionada con Z = -22,286.40 Bs.

Codigo de prueba (PHPUnit o script independiente):

<?php
// test_caso_base.php - Prueba del caso base del modelo matematico
require_once __DIR__ . '/vendor/autoload.php';
require_once __DIR__ . '/config/database.php';
require_once __DIR__ . '/app/Services/SimplexSolver.php';

use App\Services\SimplexSolver;

$solver = new SimplexSolver();

$resultado = $solver->resolver([
    'total_cabezas' => 60,
    'Mi' => [
        'R01' => -401.44,
        'R02' => -490.28,
        'R03' => -371.44,
        'R04' => -460.28,
    ],
    'bloqueos' => ['R02' => true],  // Solo R02 bloqueada
    'tiempos' => ['R01' => 6.5, 'R02' => 9.0, 'R03' => 11.0, 'R04' => 10.0],
    'hora_salida' => '20:00',
    'prob_lluvia' => 0.10  // Sin alerta
]);

// Verificaciones
$paso = 0;
$total = 5;

echo "=== PRUEBA CASO BASE ===\n";

// 1. Factibilidad
if ($resultado['factible'] === true) {
    echo "[PASO " . ++$paso . "/$total] Factible: OK\n";
} else {
    echo "[FALLO " . ++$paso . "/$total] Se esperaba factible=true\n";
}

// 2. Ruta elegida
if ($resultado['ruta_elegida'] === 'R03') {
    echo "[PASO " . ++$paso . "/$total] Ruta elegida R03: OK\n";
} else {
    echo "[FALLO " . ++$paso . "/$total] Se esperaba R03, se obtuvo: " . $resultado['ruta_elegida'] . "\n";
}

// 3. Asignacion
if (($resultado['asignacion']['R03'] ?? 0) === 60) {
    echo "[PASO " . ++$paso . "/$total] Asignacion 60 cabezas a R03: OK\n";
} else {
    echo "[FALLO " . ++$paso . "/$total] Se esperaba 60 cabezas en R03\n";
}

// 4. Z total
$zEsperada = -22286.40;
$zObtenida = $resultado['Z'];
if (abs($zObtenida - $zEsperada) < 0.01) {
    echo "[PASO " . ++$paso . "/$total] Z = {$zObtenida} (esperado {$zEsperada}): OK\n";
} else {
    echo "[FALLO " . ++$paso . "/$total] Z esperado {$zEsperada}, obtenido {$zObtenida}\n";
}

// 5. Rutas no bloqueadas disponibles
if (count($resultado['detalle_rutas']) === 4) {
    echo "[PASO " . ++$paso . "/$total] 4 rutas evaluadas: OK\n";
} else {
    echo "[FALLO " . ++$paso . "/$total] Se esperaban 4 rutas evaluadas\n";
}

echo "\nResultado completo:\n";
print_r($resultado);

Escenario: Ruta con tiempo de viaje 12 horas y salida a las 22:00.
  hora_salida: 22:00
  tiempo_viaje: 12 horas
  hora_llegada: 22:00 + 12h = 10:00 (dia siguiente)
  Limite: 08:00
  Resultado: 10:00 > 08:00 => Ruta DESCARTADA

Codigo de prueba:

<?php

require_once __DIR__ . '/app/Services/SimplexSolver.php';

use App\Services\SimplexSolver;

$solver = new SimplexSolver();

// Caso: salida 22:00, ruta R03 con 12h -> llega 10:00, debe descartarse
$resultado = $solver->resolver([
    'total_cabezas' => 60,
    'Mi' => [
        'R01' => -401.44,
        'R02' => -490.28,
        'R03' => -371.44,
        'R04' => -460.28,
    ],
    'bloqueos' => [],
    'tiempos' => ['R01' => 6.5, 'R02' => 9.0, 'R03' => 12.0, 'R04' => 10.0],
    'hora_salida' => '22:00',
    'prob_lluvia' => 0.10
]);

echo "=== PRUEBA RESTRICCION HORARIA ===\n";
echo "Salida: 22:00, R03 tiempo: 12h (llega 10:00)\n";
echo "Limite: 08:00\n\n";

$rutaDescarte = $resultado['detalle_rutas']['R03'] ?? [];
if (!empty($rutaDescarte) && $rutaDescarte['disponible'] === false) {
    echo "[PASO 1/2] R03 descartada por horario: OK\n";
    echo "Motivo: " . $rutaDescarte['motivo_descarte'] . "\n";
} else {
    echo "[FALLO 1/2] R03 deberia estar descartada por horario\n";
}

// Verificar que R01 y R04 siguen disponibles (llegan antes de 08:00)
$r01Ok = ($resultado['detalle_rutas']['R01']['disponible'] ?? false);
$r04Ok = ($resultado['detalle_rutas']['R04']['disponible'] ?? false);
if ($r01Ok || $r04Ok) {
    echo "[PASO 2/2] Otras rutas disponibles: OK\n";
} else {
    echo "[FALLO 2/2] Deberia haber al menos una ruta disponible\n";
}

echo "\nResultado:\n";
print_r($resultado);

// Segunda prueba: si todas las rutas exceden el limite, factible=false
$resultado2 = $solver->resolver([
    'total_cabezas' => 60,
    'Mi' => ['R01' => 100, 'R02' => 100, 'R03' => 100, 'R04' => 100],
    'bloqueos' => [],
    'tiempos' => ['R01' => 20, 'R02' => 20, 'R03' => 20, 'R04' => 20],
    'hora_salida' => '22:00',
    'prob_lluvia' => 0.10
]);

echo "\n=== PRUEBA: TODAS LAS RUTAS EXCEDEN LIMITE ===\n";

    echo "[PASO] factible=false correctamente: OK\n";

    echo "[FALLO] Deberia ser factible=false\n";
}
echo "Mensaje: " . $resultado2['mensaje'] . "\n";

8.3 MANUAL BREVE DE USO
--------------------------------------------------------------------------------

1. INICIO DE SESION:
   - Acceder a http://localhost/agaprova-dss/login
   - Ingresar credenciales: usuario 'operador', password 'admin123'
   - Se recomienda cambiar la contrasena en produccion.

2. REGISTRO DE DATOS SEMANALES (orden recomendado):
   a) LOTE: Registrar el lote de ganado (cabezas, peso, condicion, estacion, hora salida)
   b) PRECIOS: Actualizar precios de Santa Cruz y Cochabamba
   c) CLIMA: Ingresar probabilidad de lluvia en Abapo
   d) BLOQUEOS: Activar/desactivar bloqueos de rutas
   e) COSTOS: Registrar costos de flete para las 4 rutas

3. CALCULO DE OPTIMIZACION (SIMPLEX):
   - Ir a "Simplex" en el menu
   - Revisar los datos cargados automaticamente
   - Hacer clic en "Calcular Optimizacion"
   - El sistema mostrara la ruta seleccionada, margenes y Z total

4. SIMULACION DE ESCENARIOS:
   - Ir a "Simular" en el menu
   - Seleccionar un escenario predefinido (A-F) o crear personalizado
   - Modificar parametros y ejecutar
   - Comparar resultados sin afectar datos reales

5. GENERACION DE REPORTE PDF:
   - Despues de calcular el Simplex, ir a "Reportes"
   - Clic en "Generar Reporte PDF"
   - Descargar o visualizar el documento generado

6. CIERRE DE SESION:
   - Clic en el icono de salida en la esquina superior derecha
   - Confirmar la accion

INTERPRETACION DE RESULTADOS:
  - Z positivo: Ganancia neta semanal
  - Z negativo: Perdida neta (el modelo minimiza la perdida)
  - "NO FACTIBLE": Ninguna ruta disponible, revisar bloqueos y clima
  - La ruta con mayor Mi (menos negativa) recibe todas las cabezas

8.4 MANTENIMIENTO Y RESPALDO
--------------------------------------------------------------------------------

RESPALDO DE BASE DE DATOS:
  mysqldump -u root -p dss_agaprova > respaldo_dss_$(date +%Y%m%d).sql

RESTAURACION:
  mysql -u root -p dss_agaprova < respaldo_dss_20260601.sql

ACTUALIZACION DE DEPENDENCIAS:
  cd /ruta/del/proyecto
  composer update

LOGS:
  Los errores se registran en el log de errores de PHP.
  Para habilitar logging especifico, configurar en database.php:
  define('APP_ENV', 'development');

SEGURIDAD RECOMENDADA:
  - Cambiar password del usuario 'operador' en produccion
  - Usar conexion HTTPS en produccion
  - Configurar firewall para restringir acceso a MySQL
  - Mantener PHP y dependencias actualizados
  - Realizar respaldos semanales de la base de datos

SOLUCION DE PROBLEMAS COMUNES:
  - Error 500: Verificar config/database.php y permisos de storage/
  - Pagina en blanco: Verificar error_log de PHP
  - Dompdf no genera PDF: Verificar que la extension gd este habilitada
  - mod_rewrite no funciona: Verificar configuracion de Apache
  - No se conecta a MySQL: Verificar credenciales y servicio MySQL
