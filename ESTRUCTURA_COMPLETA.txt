ENDPOINTS API REST (8 TOTAL)
═══════════════════════════════════════════════════════════════════════════════

Ubicación: /api/index.php

1. GET /api/?action=get_precios
   → Obtiene lista de precios activos

2. GET /api/?action=get_rutas
   → Obtiene lista de rutas disponibles

3. GET /api/?action=get_estaciones
   → Obtiene lista de estaciones climáticas

4. GET /api/?action=get_clima&estacion_id=X
   → Obtiene datos climáticos de una estación

5. POST /api/?action=validar_bloqueo
   → Valida si una ruta está bloqueada en una fecha

6. POST /api/?action=calcular_flete
   → Calcula costo de flete para una ruta y cantidad

7. GET /api/?action=listar_simulaciones
   → Lista las simulaciones del usuario actual

8. GET /api/?action=obtener_simulacion&id=X
   → Obtiene detalles de una simulación específica
